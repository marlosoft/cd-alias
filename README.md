cd-alias bash extension
=======================

## Description

Creates aliases for your `cd` bash command.
![cd alias](screenshot.png)


## Installation

```bash
git clone https://github.com/marlosoft/cd-alias.git ~/.cd-alias
cd ~/.cd-alias/
bash ./install.sh
```
